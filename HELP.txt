NtOS is a Debian live system with encrypted persistence by default.

USAGE

  $ wget -O /tmp/ntos-image.tar.gz https://github.com/ntrrg/ntos/releases/download/w50/ntos-image-w50-x64.tar.gz
  $ mkdir /tmp/image
  # tar -xf /tmp/ntos-image.tar.gz -C /tmp/image
  $ EDITOR config.mk
  # make deps-install install

BUILD

  Simple:

    $ EDITOR config.mk
    # make
    # make install

  Advanced:

    $ EDITOR config.mk
    # make deps
    # make rootfs
    # make login
    # [NO_DEBIAN_INSTALLER=true] [ISO_URL=CUSTOM_ISO_URL] make image
    # [NO_PERSISTENCE=true] make install
