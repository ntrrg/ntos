NtOS is a Debian live system with encrypted persistence by default.

Basic usage:

  $ EDITOR config.mk
  # make
  # make install

Advanced usage:

  $ EDITOR config.mk
  # make deps
  # make rootfs
  # make login
  # make rootfs-clean
  # [NO_DEBIAN_INSTALLER=true] [ISO_URL=CUSTOM_ISO_URL] make image
  # [NO_PERSISTENCE=true] make install
