**NtOS** is a Debian live system with encrypted persistence, it is attached to
the Debian testing release cycle, so there are no version numbers, no release
names or backward compatibility promises.

## Usage

```shell-session
# apt install make
```

```shell-session
$ make help
```

See <https://nt.web.ve/en/projects/ntos>.
